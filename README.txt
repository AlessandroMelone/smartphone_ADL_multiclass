This repository is organized in the following way: 

- folder "dataset": contains the dataset
	- folder "dataset_documentation": contains the dataset documentation
	
- folder "code": 	contains the python code that uses the scikit-learn library to fit different estimators to the considered dataset,
									contains both the files .py used for the hyperparameter tuning via cross-validation and the used for plotting the results   
	- folder "fitted_model":	contains the results of the hyperparameter tuning

- folder "images": contains the images used in the technical report and obtained via the files .py in the folder "code"
